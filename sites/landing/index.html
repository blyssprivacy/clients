Spiral

Spiral is a database that cannot learn which rows you read.

You can try using to read Wikipedia without revealing to the server which articles you read, or look up the balance of a Bitcoin address without revealing the address.

The Spiral scheme is outlined in our paper "Spiral: Fast, High-Rate Single-Server PIR via FHE Composition", which was published at IEEE S&P'22 ("Oakland"). The code for the base version of Spiral is available here.

We are building a fully managed version of our private database that you'll be able to use as a complete service. Sign up for our mailing list here. If you have a use case you're interested in chatting with us about, please email menon.samir@gmail.com.
